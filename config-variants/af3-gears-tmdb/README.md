# af3-gears-tmdb

Gears on AF3 with TMDb service-widgets (crash mitigation, mirrors
zephyr-gears-tmdb). Derived from af3-pov-tmdb: plugin.video.pov->gears, the POV
player (play_media) -> gears.json (playback.meta), Connect Services (POV
navigator.db) -> TorBox Services (gears navigator.db via config_policy
gears_shortcuts). Delivered as the Gears AF3 default via config/.
